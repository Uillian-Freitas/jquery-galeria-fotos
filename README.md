# 🖼️ Galeria de Fotos (jQuery)

Galeria de imagens desenvolvida para exibição de fotos em página web, utilizando HTML, CSS e JavaScript com jQuery para manipulação e interação.

## 🚀 Funcionalidades

* Exibição de imagens organizadas em galeria
* Manipulação de elementos com JavaScript
* Estilização visual com CSS
* Estrutura separada em arquivos de marcação, estilo e script

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* jQuery

## 📂 Estrutura do Projeto

```bash
jquery-galeria-fotos/
├── images/
├── index.html
├── main.css
└── main.js
```

## ▶️ Como Executar

Abra o arquivo `index.html` no navegador.
